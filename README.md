# Psicólogo Virtual Unicórdoba

Sistema de asistencia psicológica virtual que combina análisis de emociones con IA conversacional para brindar apoyo empático y personalizado.

## Tecnologías

- **Frontend**: Next.js + React
- **Backend**: Spring Boot + FastAPI
- **IA**: Llama 3 + clasificador de emociones
- **Base de datos**: PostgreSQL
